## Issue

You published a documentation page, but it isn't showing up.

## Quick Fix

<Steps>
  <Step title="Check page status">
    Make sure the page is marked as **Published** (not 'Draft').
  </Step>
  <Step title="Refresh the dashboard">
    Press Ctrl+R or use the browser reload button.
  </Step>
  <Step title="Verify category">
    Confirm the page is assigned to a visible category.
  </Step>
  <Step title="Try logging out and back in">
    Sometimes a new session helps load updates.
  </Step>
</Steps>
